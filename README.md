# 🤖 Crown Core Discord Bot

Crown Core is a powerful Discord bot built for server automation, activity tracking, moderation, and utility features. It is designed to be lightweight, scalable, and easy to deploy on platforms like Railway.

---

## 🚀 Features

- 📊 Activity tracking system
- 🏆 Role & level system
- ⚙️ Moderation commands
- 📢 Automated announcements
- 🔐 Secure environment variable configuration
- 🟢 24/7 hosting support (Railway / VPS ready)
